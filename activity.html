Activity Manager


The Story Behind

Department Activity Manager is the efficient handling of activities of a particular department especially for a college. The project aims at creating an interface to list activities matching particular criteria with relative ease in an efficient manner

The Project database contains information about various activities related to academics, sports, extracurriculars, conferences , seminars and so on. It is one portal where people can register for activities with just few clicks. Constant notifications will be sent to the registered members. Changes to the database can only be made by the Administrator or a staff member with proper rights. The project has a very attractive interface and no prior programming knowledge is necessary use the project. Its smart search bar accepts English statements queries and displays the required results back immediately. An option of input by voice is also available where the user can speak the quires into a microphone and the the query is understood by the computer.
